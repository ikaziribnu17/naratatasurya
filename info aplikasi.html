🚀 ASTRO-QUEST (Versi 1.0)

ASTRO-QUEST adalah purwarupa aplikasi pembelajaran interaktif berbasis web (Web-App) yang dirancang khusus untuk meningkatkan motivasi dan pemahaman siswa Kelas VII SMP pada materi Sistem Tata Surya.

Aplikasi ini mengusung konsep Gamifikasi, di mana siswa bertindak sebagai "Kadet/Kapten" yang harus menavigasi ruang materi, melakukan eksperimen di Laboratorium Simulasi, hingga menempuh Misi Final untuk mendapatkan Sertifikat Kelulusan.

🎨 REFERENSI & KREDIT ASET

Pengembangan antarmuka kosmik ini menggunakan beberapa sumber terbuka pihak ketiga (Open Source) untuk keperluan edukasi non-komersial:

  • Latar Belakang Kosmik: Resolusi tinggi bintang dan nebula bersumber dari Unsplash.
  • Model Planet & Animasi: GIF Rotasi Bumi dan ornamen tata surya diambil dari domain publik Wikimedia Commons.
  • Efek Suara (SFX): Suara tombol kokpit, alarm, dan efek mesin pesawat (Warp Speed) disediakan oleh SoundJay.com.
  • Aset Karakter: Ikon Astro-Bot diambil dari repositori GitHub resmi pengembang @ikaziribnu17.
  • Tipografi: Menggunakan font Fredoka dan Poppins dari Google Fonts.

🛠️ TEKNOLOGI PENGEMBANGAN

Dibangun secara *Native* menggunakan standar web modern (HTML5, CSS3 Grid/Flexbox, dan Vanilla JavaScript). Dilengkapi dengan teknologi Canvas API untuk menghasilkan sertifikat digital otomatis tanpa memerlukan server tambahan.
